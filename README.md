# Youth Policy Chatbot MVP (Local)

## 목표
- 청년이 정책 문서를 읽기 힘든 문제를 해결하기 위해
- 사용자 상황을 2~3단계 질문으로 구조화 수집하고
- 정책 문서 기반으로 맞춤 추천/요약/Q&A를 제공하는 로컬 MVP

## 특징
- 주민번호/계좌/정확한 주소/잔고/자산총액 등은 묻지 않음
- 입력은 저장하지 않음(서버 메모리 임시, 재시작 시 초기화)

## 코멘트 
- 현재 버전은 '상담사'라기보다 문서 요약/정보 제공 챗봇에 가까우며, 2차 업데이트 예정
